BMI Calculator
Welcome to the BMI Calculator! This is a simple Python program that calculates your Body Mass Index (BMI) based on your height and weight inputs.

Getting Started

To calculate your BMI, follow these steps:

1. Run the program.
2. When prompted, enter your height in meters.
3. When prompted, enter your weight in kilograms.
4. The program will calculate your BMI using the formula: BMI = weight / (height^2).
5. It will display your calculated BMI.

Prerequisites
Python 3.x

Usage

1. Open a terminal or command prompt.
2. Navigate to the directory where the code file is located.
3. Run the following command to execute the program:python bmiCalculator.py

Example

Enter your height in m: 1.75
Enter your weight in Kg: 65
As per your Height and Weight, your BMI is: 21.22

Please note that the BMI value is an indicator of body fatness and is used to screen for weight categories that may lead to health problems. Consult a healthcare professional to interpret your BMI value accurately.
